CSS Flex Demonstration

display:flex puts child elements in a one-dimensional (e.g. x- or y-axis) container. This means flex containers behave as rows or columns.

Examples

First we define a div that will act as the parent (row or column). Keep in mind that the class names here are mentioned only to identify where particular rules are applied; they are not part of any standard.
